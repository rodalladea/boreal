# Boreal

A minimal floating camera overlay for macOS. It shows your webcam feed in a small window that stays on top of everything — perfect for screen recordings.

## Features

- Floating window that stays on top of all apps and spaces
- Switch between multiple cameras via the menu bar
- Draggable — move it anywhere on your screen
- No window chrome, just your camera with rounded corners
- Automatically repositions when your display setup changes
- Detects cameras being plugged in or unplugged

## Requirements

- macOS 14.0+
- A camera (built-in or external)

## Installation

1. Clone the repository
2. Open `Boreal.xcodeproj` in Xcode
3. Build and run (Cmd+R)

## Usage

Launch the app and the camera overlay appears in the top-right corner of your screen. Drag it wherever you want.

Use the **Camera** menu in the menu bar to switch between cameras (Cmd+1, Cmd+2, etc.).
